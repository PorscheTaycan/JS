        #DOM객체를 JS코드로 식별하는 방법
        #document객체의 메서드/컬렉션/프로퍼티

        #프로퍼티
        location : 현재 문서의 URL 정보를 가진 객체
        readyState : 문서의 현재 로딩 상태
        title : 문서 제목
        head : 헤드 객체
        body : 바디 객체
        activeElement : 현재 포커스된 요소

        #컬렉션
        image : 문서 내 img객체들의 컬렉션
        links : 문서 내 링크 객체들 컬렉션
        forms : 문서 내 폼 객체 컬렉션


        #메서드
        document.getElementByID('a') : ID로
        document.getElementsByClassName('b') : 클래스명
        document.getElementsByName('c') : name속성으로
        document.getElementsByTagName('div') : 태그명
        addEventListner() : document객체에 이벤트 리스너 추가
        close() : document 객체에 HTML 컨텐츠를 브라우저에 전부 출력하고 더이상 쓰기를 받지 않음.
        createElement() : HTML 태그의 동적 생성
        removeEventListener() : 이벤트 리스너 제공하거나
        write() : 쓴다.
        whiteln() : 쓴다. (줄바꿈 된다.)
